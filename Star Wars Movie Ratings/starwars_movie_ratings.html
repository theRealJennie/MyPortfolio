Star Wars Movie Ratings

For this project I used Python to scrape data from Rotten Tomatoes, then used Tableau to analyze how well the critics and fans liked each movie.

I coded the Python part in VSCode using a MovieScraper library. I then saved that into a .csv file, which I then used Tableau to manipulate and analyze.
